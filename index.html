Phoenix GameEngine Demo

Demo instructions

  • On mobile / tablets: watch demo only
  • On laptop / desktops: Click in watch window to give game focus then use the left and right arrow keys to position tank turret
  • Refresh page to restart demo

Engine features

  • Designed for use on web and Tizen platform (Samsung Gear watches)
  • Rendering using hardware accelerated WebGL
  • Audio playback using audiosprites and WebAudio
  • Animations and images packaged using texture atlas / sprite sheet
  • Animations are controlled using controller objects
  • Animations playback adjust to machine performance
  • Support for background tiles
  • Object Collision detection
  • Bitmap font support used by text sprites
  • Dedicated math library for trig functions
  • Managers:
    • Asset preloader
    • Game preloader, data driven using a game.json file
    • Projectile manager
    • Event manager
  • Built with TypeScript 3

Additional Info

Engine is under development and being used as the basis for an upcoming game called Tyros
